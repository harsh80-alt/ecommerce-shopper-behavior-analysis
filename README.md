# E-Commerce Shopper Behavior & Lifestyle Analysis

## Project Overview
This Power BI project analyzes e-commerce shopper behavior, customer demographics, spending patterns, and lifestyle trends using interactive dashboards and visual analytics.

---

## Objectives
- Understand customer purchasing behavior
- Analyze demographic and lifestyle patterns
- Identify high-spending customer segments
- Track monthly and weekly spending trends
- Analyze return rate behavior

---

## Key Insights
- 1 Million Total Users Analyzed
- 255M Average Order Value
- 2499M Monthly Spending
- 10M Weekly Purchases
- 50% Average Return Rate
- Urban segment contributed highest spending

---

## Dashboard Features
- Customer Segmentation Analysis
- Country-wise User Distribution
- Monthly Spending Trends
- Demographic Insights
- Lifestyle Analysis
- Return Rate Monitoring
- Interactive Filters & Visuals
